# Modality-Adaptive Multimodal Instance-Graph Learning for Intangible Cultural Heritage Classification

## Abstract:
The digitization of intangible cultural heritage (ICH) has increased the need for automatic content analysis, yet ICH recognition remains challenging because discriminative evidence is often distributed across visual scenes, subtitle text, and audio. Visual-only models are therefore often insufficient for categories that share similar costumes, crowds, stages, or ritual layouts. Existing multimodal ICH studies have shown that text can improve recognition, but they still rely largely on flat fusion and provide limited support for partial modality availability.
We present MAMIG, a modality-adaptive multimodal instance-graph learning framework for clip-level ICH classification. In MAMIG, each clip-derived instance is represented as a heterogeneous graph whose nodes correspond to observed image, text, and audio elements. The framework combines modality-specific encoding, shared latent projection, learnable edge weighting, weighted graph-attention propagation, and graph-level prediction with either an MLP or a FastKAN head. It also includes end-to-end training and modality-adaptive inference procedures that construct each graph directly from the observed modalities, without introducing synthetic placeholders for missing inputs.
We evaluate MAMIG on a curated Vietnamese ICH multimodal dataset and compare it against strong baselines under a shared frozen-encoder benchmark with a grouped source-video split. The best configuration achieves 94.35\% accuracy and 94.38\% Macro F1 Score, improving Macro F1 Score by 4.32 percentage points over the strongest image-only baseline and by 1.90 percentage points over the strongest non-graph multimodal baseline. Ablation and sensitivity analyses show that learnable edge weighting, structure-aware readout, modality embeddings, and modality-adaptive inference each contribute to performance. These findings support structure-aware multimodal reasoning for clip-level ICH analysis.
